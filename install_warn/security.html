Security Concerns with the Exponent Install Wizard

The Exponent Install Wizard guides you through the installation, configuration and setup of an Exponent site. It is provided to make installing and using the Exponent CMS easier and simpler. However, with ease-of-use comes decreased security.

Because the Installation Wizard modifies the sites configuration file, Exponent will not allow the site to be used until the Wizard is disabled. Otherwise, anyone could visit the installer and modify crucial configuration data.

Disabling the wizard can be done two ways: deleting the install/ directory or changing its permissions so that the web server cannot read it.

Removing the Wizard Directory

This is the simplest method of disabling the wizard. On UNIX, this can be done through FTP (if your web host provides FTP access) or through Secure Shell or Telnet (provided that you have shell access to the web server).

Changing Permissions

If you want to keep the installation directory around, you need to render it inaccessible to the system user running the web server. This can be done either through an FTP client that supports the CHMOD command (like WS_FTP, or any command-line ftp client), or through the shell.

To be on the safe side, change the permissions so that no one can read or write to the install/ directory. The octal number for this permission setting is '000'.
